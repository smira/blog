Тест
----

После вебинара Вы можете пройти небольшой тест для лучшего запоминания
полученной информации. Прохождение теста не является обязательным. В
ответ на почту будут отправлены результаты тесты и мои комментарии.

Итак, `тест по итогам вебинара №6 <https://docs.google.com/forms/d/1wV0FQF7UJYEHWWI13K2Yoz1sdB2_l6qDaCFG6nrlJfI/viewform>`_.


Практические задания
--------------------

Для выполнения практических заданий необходимо `установить Docker <https://docs.docker.com/installation/#installation>`_.
Для Windows/Mac OS X будет установлена отдельная виртуальная машина с Linux, для Linux доступна
установка в рамках существующей OS.

После установки Docker необходимо скачать контейнер с заданиями (800-900Мб)::

    docker pull smira/hl-tasks:base

В этом вебинаре было 1 задание:

1. `Эффект от использования “обычного” и exponential backoff на стороне клиента. <https://github.com/smira/hl-tasks/blob/task-exponential/description.pdf?raw=true>`_ (`подробное описание на Хабре <http://habrahabr.ru/post/227225/>`_)
