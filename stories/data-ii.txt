Практические задания
--------------------

Для выполнения практических заданий необходимо `установить Docker <https://docs.docker.com/installation/#installation>`_.
Для Windows/Mac OS X будет установлена отдельная виртуальная машина с Linux, для Linux доступна
установка в рамках существующей OS.

После установки Docker необходимо скачать контейнер с заданиями (800-900Мб)::

    docker pull smira/hl-tasks

В этом вебинаре было 3 задания:

1. `memcached и slabы <https://github.com/smira/hl-tasks/blob/task-memcached/description.pdf?raw=true>`_
2. `"сравнение" Redis и PostgreSQL <https://github.com/smira/hl-tasks/blob/task-pgredis/description.pdf?raw=true>`_
3. `консистентное хеширование на примере twemproxy <https://github.com/smira/hl-tasks/blob/task-twemproxy/description.pdf?raw=true>`_
